The following links demonstrate how one can set up a canvas element to better adapt to the client display and browser.
  • Retina displays: demonstrates the difference between a canvas that is adapted to retina displays versus one that is not.
  • Viewport scaling: demonstrates how one can scale a canvas to fill the viewport while maintaining a fixed aspect ratio.
  • Efficient animation: demonstrates how one can animate a canvas so that system resources are not wasted when the webpage is not in view. The canvas is also resized as the viewport dimensions change. This webpage also decouples the animation callback from the time-based state changes (e.g. physics updates).
